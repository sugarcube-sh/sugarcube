## Custom directories

By default, `init` places files based on whether your project has a `src/` directory:

<Table>
| | With `src/` | Without `src/` |
|---|---|---|
| Design tokens | `src/design-tokens` | `design-tokens` |
| CUBE CSS | `src/styles` | `styles` |
| Components | `src/components/ui` | `components/ui` |
</Table>

You can override any of these during `init`:

<PackageManagerTabs command="npx @sugarcube-sh/cli init --tokens ./tokens --cube ./css --components ./ui" />

For ongoing customization beyond `init`, see [Configuration](/docs/configuration).
