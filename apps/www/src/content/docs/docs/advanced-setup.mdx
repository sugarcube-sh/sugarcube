## Custom directories

By default, `init` places tokens in `src/design-tokens` (or `design-tokens` if there's no `src/` directory), and CUBE CSS in `src/styles` (or `styles` if there's no `src/` directory).

You can override both during `init`:

<PackageManagerTabs command="npx @sugarcube-sh/cli init --tokens-dir ./tokens --cube-dir ./css" />
