<SiteAside type="note">
    The card component is intentionally minimal — it's little more than a container with basic styling (border, border-radius, background, and shadow). The optional structure classes (`.card-header`, `.card-content`, `.card-footer`) only provide padding and layout hints. For colors, spacing, typography, and other visual styling, use utility classes or your own custom CSS. This keeps the card flexible and adaptable — a starting point for your own card component.
