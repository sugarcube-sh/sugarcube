## Usage

<SiteAside type="note">
    Native dialogs use the `showModal()` method to open and `close()` method to close. You can also use forms with `method="dialog"` for form-based dialogs.
</SiteAside>

Apply the `.dialog` class to a native `dialog` element:

```html
<dialog class="dialog" id="my-dialog">
  <h2>Dialog Title</h2>
  <p>Dialog content goes here.</p>
  <button onclick="this.closest('dialog').close()">Close</button>
</dialog>

<button onclick="document.getElementById('my-dialog').showModal()">
  Open Dialog
</button>
```
