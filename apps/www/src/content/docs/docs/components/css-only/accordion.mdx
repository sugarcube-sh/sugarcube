## Usage

The styles are designed to work with native `details` and `summary` elements. Apply the `.accordion` class to a container:

```html
<div class="accordion">
  <details>
    <summary>Section 1</summary>
    <div class="accordion-content">
      <p>Content for section 1</p>
    </div>
  </details>
  
  <details>
    <summary>Section 2</summary>
    <div class="accordion-content">
      <p>Content for section 2</p>
    </div>
  </details>
</div>
```
