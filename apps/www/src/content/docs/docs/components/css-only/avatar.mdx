## Usage

Apply the `.avatar` class to display an avatar with an image or fallback:

```html
<span class="avatar">
  <img src="/avatar.jpg" alt="User" />
</span>

<!-- With fallback initials -->
<span class="avatar">
  <span class="avatar-fallback">AB</span>
</span>
```
