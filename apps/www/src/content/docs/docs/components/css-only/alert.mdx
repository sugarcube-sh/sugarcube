## Usage

Apply the `.alert`, `.alert-title`, and `.alert-description` classes to your markup:

```html
<div class="alert">
  <svg><!-- icon --></svg>
  <div class="alert-title">Alert Title</div>
  <div class="alert-description">Alert description text</div>
  <button class="button">Action</button>
</div>
```
