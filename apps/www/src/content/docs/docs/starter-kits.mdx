Starter kits are pre-built design token sets that are designed to provide a solid foundation for your project. When you run `init` without existing tokens, sugarcube prompts you to choose a starter kit.

## Available Kits

### Fluid

Uses [`fluidDimension`](/docs/fluid-space-and-type) tokens for spacing and typography that scale with viewport size using CSS `clamp()`.

### Static

Uses fixed dimensions (px/rem).

## Customization

Starter kits are starting points, not constraints. Edit the JSON files to build your design system.
