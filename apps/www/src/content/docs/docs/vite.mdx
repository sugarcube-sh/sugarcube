## Create your project

Start by creating a new Vite project if you don't have one set up already. The most common approach is to use [Create Vite](https://vitejs.dev/guide/#scaffolding-your-first-vite-project).

<PackageManagerTabs command="npm create vite@latest my-project" />

Then navigate into your project:

```bash frame="none"
cd my-project
```

## Initialize sugarcube

<PackageManagerTabs command="npx @sugarcube-sh/cli init" />

## With the Vite plugin

If you chose the Vite plugin during `init`, add it to your configuration (you may have to create `vite.config.{ts|js}` if it doesn't exist):

```ts title="vite.config.ts"
import { defineConfig } from "vite";
import sugarcube from "@sugarcube-sh/vite";

export default defineConfig({
    plugins: [sugarcube()],
});
```

Then import the virtual CSS module in your project's entrypoint:

```js title="src/main.js"
import "virtual:sugarcube.css";
```

## Without the Vite plugin

If you skipped the Vite plugin during init, [generate CSS with the CLI](/docs/reference/cli-commands#generate) and import the files directly:

```js title="src/main.js"
import "./styles/global/tokens.variables.gen.css";
```
