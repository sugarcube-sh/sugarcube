## Create your project

Start by creating a new TanStack Start project if you don't have one set up already:

<PackageManagerTabs command="npm create @tanstack/start@latest" />

## Initialize sugarcube

<PackageManagerTabs command="npx @sugarcube-sh/cli init" />

## With the Vite plugin

If you chose the Vite plugin during `init`, add it to your Vite configuration. **Important**: The sugarcube plugin must run before `tanstackStart()`:

```ts title="vite.config.ts"
// other imports
import sugarcube from "@sugarcube-sh/vite";

export default defineConfig({
    plugins: [
        devtools(),
        nitro(),
        viteTsConfigPaths({
            projects: ["./tsconfig.json"],
        }),
        // sugarcube must run before TanStack Start to process virtual CSS
        sugarcube(),
        tanstackStart(),
        viteReact(),
    ],
    build: {
        // Workaround for UnoCSS multi-environment issue
        // See: https://github.com/unocss/unocss/issues/4990
        outDir: ".output/public",
    },
});
```

Then import the virtual CSS module in your root component:

```tsx title="src/routes/__root.tsx"
import "virtual:sugarcube.css";
```

## Without the Vite plugin

If you skipped the Vite plugin during init, generate CSS with the CLI:

<PackageManagerTabs command="sugarcube generate" />

TanStack Start requires CSS to be added via the `head` function to avoid a flash of unstyled content during SSR:

```tsx title="src/routes/__root.tsx"
/// <reference types="vite/client" />
import tokensCss from "../styles/global/tokens.variables.gen.css?url";

export const Route = createRootRoute({
    head: () => ({
        links: [{ rel: "stylesheet", href: tokensCss }],
    }),
    // ...
});
```

## If you added CUBE CSS

CUBE CSS files are source files you own — not virtual modules. Create a CSS file that imports them:

```css title="src/styles/index.css"
/* CUBE CSS - your source files */
@import "./global/global.css";
@import "./compositions/flow.css";
@import "./compositions/cluster.css";
/* ... add the files you're using */
```

Then add it to the `head` links:

```tsx title="src/routes/__root.tsx"
/// <reference types="vite/client" />
import styles from "../styles/index.css?url";

export const Route = createRootRoute({
    head: () => ({
        links: [{ rel: "stylesheet", href: styles }],
    }),
    // ...
});
```
