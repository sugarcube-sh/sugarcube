# Contributing to sugarcube

Thanks for your interest in contributing!

## Reporting Issues

- Search existing issues before opening a new one
- Include reproduction steps and environment info

## Pull Requests

- Open an issue first to discuss significant changes
- Keep PRs focused on a single change

## Development

```bash
pnpm install
pnpm build
pnpm test
```

Run the docs site locally:

```bash
pnpm dev:www
```

## Questions?

Open a [discussion](https://github.com/sugarcube-sh/sugarcube/discussions).
